# Contractor Workforce Risk Intelligence System

## Overview

A workforce analytics solution developed to identify absenteeism risks, vendor performance issues, and cost leakage among contract workers.

## Tools Used

- MySQL
- SQL
- Power BI
- DAX
- Excel

## Dataset

- 455 Workers
- 142,415 Records
- 10 Vendors

## Business Problem

Contract workforce absenteeism and vendor performance issues were causing operational inefficiencies and hidden costs.

## Key Findings

- Identified 3 high-risk vendors
- Absenteeism rates between 22%–29%
- Estimated cost leakage of ₹4.4 lakh

## Dashboard Features

- Vendor Risk Analysis
- Absenteeism Trends
- Workforce Distribution
- Cost Leakage Tracking
- Interactive Filters and Slicers
